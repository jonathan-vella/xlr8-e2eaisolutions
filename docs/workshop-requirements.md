# Workshop Requirements

## Attendee Prerequisites

This document outlines the recommended knowledge, skills, and certifications that attendees should have before participating in the "Designing an End-to-End Azure AI Solution" workshop for Innovate Financial Services (IFS).

## Microsoft Partner Specialization Alignment

This workshop is aligned with the **AI Platform on Microsoft Azure** specialization for Microsoft partners. This specialization validates a partner's deep knowledge, extensive experience, and proven success in planning and deploying Azure AI solutions involving conversational AI, natural language processing, knowledge mining, and custom AI solutions.

### About the AI Platform on Microsoft Azure Specialization

The AI Platform on Microsoft Azure specialization is designed for partners who demonstrate deep knowledge in deploying and maintaining secure, performant, and reliable Azure AI solutions. Partners who achieve this specialization have verified capabilities in:

- Designing AI solutions that follow Microsoft's Well-Architected Framework and Cloud Adoption Framework
- Implementing Azure AI services with appropriate security controls and governance
- Building end-to-end AI solutions that address real business challenges
- Operating and maintaining AI workloads with attention to performance, scalability, and reliability

This workshop provides practical experience and knowledge that directly supports the skills required for this specialization, helping organizations prepare for the associated assessments and validations.

### Specialization Benefits

Organizations pursuing this specialization will develop capabilities to:
- Design secure and scalable Azure AI foundations for enterprise customers
- Implement advanced Azure AI services including Azure OpenAI Service
- Build modern AI applications using best practices and proven architectural patterns
- Ensure proper security, governance, and compliance for enterprise AI solutions

For more information about this specialization, visit the [Microsoft Partner Center](https://partner.microsoft.com/en-us/partnership/specialized-areas).

## Required Knowledge

### Azure Fundamentals
- Azure subscription and resource management concepts
- Basic understanding of Azure Resource Manager (ARM) and resource organization
- Familiarity with Azure portal navigation and basic operations
- Knowledge of core Azure services (Compute, Storage, Networking)

### Cloud Architecture Concepts
- Understanding of common cloud architecture patterns
- Familiarity with High Availability and Disaster Recovery concepts
- Basic understanding of security principles (least privilege, defense-in-depth)
- Experience with multi-tier application architectures

### AI/ML Concepts
- Foundational understanding of artificial intelligence and machine learning concepts
- Basic knowledge of Large Language Models (LLMs) and their capabilities
- Familiarity with data processing concepts for AI/ML workloads
- Understanding of prompt engineering concepts (helpful but not required)

### Development & Operations
- Basic understanding of Infrastructure as Code (IaC) principles
- Familiarity with DevOps practices for deployment and operations
- Understanding of API design and integration patterns
- Knowledge of monitoring and observability concepts

## Technical Skills

- **Diagramming Skills**: Ability to create and interpret architecture diagrams
- **Problem-Solving Skills**: Capacity to analyze requirements and design appropriate solutions
- **Collaboration Skills**: Experience working in cross-functional technical teams
- **Communication Skills**: Ability to articulate technical concepts and design decisions

## Recommended Certifications

While not mandatory, the following Microsoft certifications provide a solid foundation for this workshop:

### Core Certifications

- [Microsoft Certified: Azure Fundamentals (AZ-900)](https://learn.microsoft.com/en-us/certifications/azure-fundamentals/) - **Strongly Recommended**
  
  This certification validates your understanding of cloud concepts, Azure services, security, privacy, compliance, and pricing.

- [Microsoft Certified: Azure Administrator Associate (AZ-104)](https://learn.microsoft.com/en-us/certifications/azure-administrator/)
  
  This certification demonstrates your ability to implement, manage, and monitor Azure environments.

### Architecture Certifications

- [Microsoft Certified: Azure Solutions Architect Expert (AZ-305)](https://learn.microsoft.com/en-us/certifications/azure-solutions-architect/)
  
  This certification validates your expertise in designing solutions that run on Azure, including compute, network, storage, and security.

- [Microsoft Certified: DevOps Engineer Expert (AZ-400)](https://learn.microsoft.com/en-us/certifications/devops-engineer/)
  
  This certification shows your ability to combine people, processes, and technologies to continuously deliver valuable products and services.

### AI/ML Certifications

- [Microsoft Certified: Azure AI Fundamentals (AI-900)](https://learn.microsoft.com/en-us/certifications/azure-ai-fundamentals/)
  
  This certification validates your knowledge of common AI and machine learning workloads and how to implement them on Azure.

- [Microsoft Certified: Azure AI Engineer Associate (AI-102)](https://learn.microsoft.com/en-us/certifications/azure-ai-engineer/)
  
  This certification demonstrates your ability to implement Azure AI solutions including natural language processing, computer vision, and conversational AI.

## Workshop Preparation

To get the most out of this workshop, we recommend the following preparation:

### Self-Study Resources

1. **Azure Landing Zones**
   - Review the [Microsoft Cloud Adoption Framework for Azure](https://learn.microsoft.com/en-us/azure/cloud-adoption-framework/)
   - Study the [Azure Landing Zone architecture](https://learn.microsoft.com/en-us/azure/cloud-adoption-framework/ready/landing-zone/)

2. **Azure AI Services**
   - Review the [Azure OpenAI Service documentation](https://learn.microsoft.com/en-us/azure/ai-services/openai/)
   - Understand the [Azure AI Search capabilities](https://learn.microsoft.com/en-us/azure/search/)
   - Explore the [Retrieval-Augmented Generation (RAG) pattern](https://learn.microsoft.com/en-us/azure/architecture/ai-ml/architecture/baseline-openai-e2e-chat)

3. **Security & Governance**
   - Study [Private Endpoints for Azure services](https://learn.microsoft.com/en-us/azure/private-link/private-endpoint-overview)
   - Understand [Azure RBAC and Managed Identities](https://learn.microsoft.com/en-us/azure/active-directory/managed-identities-azure-resources/)
   - Review [Azure Policy for governance](https://learn.microsoft.com/en-us/azure/governance/policy/)

### Hands-on Experience

The workshop will be more valuable if you have hands-on experience with:

- Creating and configuring Azure resources through the portal and IaC
- Working with Azure Virtual Networks and subnets
- Deploying PaaS services with private networking
- Basic experience with Azure OpenAI and AI Search
- Using Azure Key Vault for secrets management

## Technical Requirements

- Laptop with reliable internet connectivity
- Modern web browser (Microsoft Edge, Chrome, Firefox)
- Microsoft account with access to Azure portal
- Optional: Visual Studio Code with Azure extensions installed
